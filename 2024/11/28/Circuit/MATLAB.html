算术运算符

符号 作用
+和- 矩阵中只要兼容就可加，五种兼容模式：大小相同、一方为标量常数、行数相同列向量、列数相同行向量、列向量±行向量
* 和.* *就是正常的矩阵乘法，.*是对应元素相乘
/和\和.\ /左除（求A*x=B的x），\右除，.\对应元素相除
和. ^矩阵幂运算，. ^对应元素幂运算

关系运算符：

==、~=(不等于)、>、<、>=、<=，对于矩阵符合兼容模式算法，返回logical元素矩阵

逻辑运算符

（and()）&、（or()）|、（not()）~、xor()（比较逻辑值，比如xor(3,4)结果为0），对于矩阵符合兼容模式算法，返回logical元素矩阵
&&、||与&、|相同但只用于标量且具有短路功能

矩阵

创建矩阵

  • A=[a,b,c]创建行向量（，可换空格）

  • A=[a;b;c]创建列向量（；可换换行）

  • A=[a b c;e f g]创建多维矩阵

  • A=a:b:c,即从a到c,步长为b的矩阵

  • 用函数

索引

MATLAB索引从1开始。引用格式：

  • A(row_ind,col_ind)

  • A([a,b,c...])

  • A(a:b:c)（索引多个数据用）

  • 用end关键字

  • A(:,a)返回第a列所有元素

  • A(A>0)返回A中大于0的元素（逻辑值引用）

  • linspace()

修改删除元素

  • A(1)=0

  • A(a:b:c)=[]注意左右元素个数要相等

  • A(a:b:c)=x将所有选中索引全修改为一个常数

  • A(...)=[]删除

程序流程控制

一、条件语句

if-elseif-else-end语句：

if x>=90 && x<=100 %不需要分号或冒号
    dj=1
elseif x<90 && x>=60
    dj=2
else
    dj=3
end %有if就一定要有end
dj

在MATLAB中可以用ctrl+i来格式化选中代码

switch-case-otherwise-end语句

season=randi([1,4])
switch season
    case 1
        disp("春季");
    case 2
        disp("夏季");
    case 3
        disp("秋季");
    otherwise
        disp("冬季");
end

如果case后跟的是元胞数组，只要开关变量和数组中任意一个元素相等即可

二、循环语句

for-end语句

for ii=[3;4;5;6;7]%向量或矩阵，如果是矩阵，遍历的元素是每一列向量
    %循环体
end

还可以：

for ii=3:7
    %循环体
end

while-end语句

m=1;
while y<=10
    m=m+1;
end

注意：如果死循环后可以按ctrl+c退出

break和continue

和C一样，不赘述

三、其他

1.try catch语句

try的代码块是可能引发故障的语句，若故障，则执行catch块的代码

try
    A+B
catch
    disp("大小不兼容")
end

2.tic/toc

可以输出tic和toc之间程序的运行时长

tic
sum=0;
for i=1:7
    sum=sum+i;
    disp(sum);
end
toc

命令行窗口会自动输出运行时间

3.pause

  • pause:使程序停止，待用户按下任意键继续

  • pause(n):使程序停止n秒

4.input

  • input(‘提示’),如果要输入字符串需要自己加引号

  • input(‘提示’，‘s’)，不需要自己加引号，专门接受文本

5.warning/error

warning(“txt”)：自定义生成警告文本
warning on/off：开启或关闭警告功能
error(“txt”)：自定义生成报错文本
区别：warning的信息是橘红色，error是红色；warning后的代码还会运行，error直接退出程序

元胞数组

用于存储不同数据类型的数据容器，用{}括起来，形式上与矩阵相似，可以储存矩阵甚至元胞数组

常见数据类型：

  • 数值：包括整型和浮点

  • 文本：包括字符（char）和字符串（string）

  • 逻辑（logical）：布尔变量true/false

  • 元胞数组（cell）

  • 结构体数组（struct）：

  • 分类数组（categorical）：保存分类型变量

  • 日期时间（datetime）：保存日期、时间的数据

  • 表格（table）：保存表格形式数据

  • 函数句柄：可引用和间接调用函数

查看变量类型可以用class(a)查看

形式和索引查找一样，但不是直接返回该元素，而是返回一个对应大小的元胞数组

{}引用

形式和索引查找一样，直接返回该元素

cc{1}[2] %对应的即是2
cc{2}[3] %对应的就是c
